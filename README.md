# Kotlin Multiplatform Template  
### Android ‚Ä¢ iOS ‚Ä¢ Desktop ‚Ä¢ Web (KMP + Compose Multiplatform)

A production-ready **Kotlin Multiplatform (KMP)** project template designed to help you quickly start building apps for **Android, iOS, Desktop, and Web** from a single shared codebase.

---

## Features

-  **Compose Multiplatform UI** for all platforms  
-  **Android app** (Jetpack + Koin + Ktor + Room)  
-  **iOS app** (Swift integration, KMM framework)  
-  **Desktop app** (Compose Desktop + JVM)  
-  **Web app** (Compose Web + WASM/JS)  
-  **Ktor Multiplatform Networking**  
-  **Room DB (Android)**  
-  **Common unit tests**  
-  **Modular project structure**  
-  **Gradle Version Catalog + Kotlin DSL**
-  **All Have Share UI**
